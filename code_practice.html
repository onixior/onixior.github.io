the origins...

available for anybody to use regardless of their computer, the terminal allows for an endless amount of opportunities to directlycommunicate with the computer. it's essentially an input-based program that uses code as its primary way of interpretation and interaction.

why we use the terminal.

  • we can automate repetitive tasks and processes.
  • we can use it to develop software such as html!
  • it's super convienent and is already a part of the computer.
  • we can have more access to commands and provide shortcuts for navigation.

wow, this sounds super cool!
what specific commands can we use?

command purpose
mkdir make a directory within the desktop or repository.
cd move to directory.
vim enable shortcuts and navigation easier. we can also make files with :wq.
cat / less either can be used to open file contents

these aren't the only commands however!
research and experiment with more, with knowing these basic navigation tools.
